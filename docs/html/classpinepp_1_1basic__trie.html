Detailed Description

template<char_type T = char>
class pinepp::basic_trie< T >

Template class for storing strings without duplicates.

A basic_trie is a structure that allows for constant time lookup and insertion of strings (linear in the size of the inserted/searched string without duplicates. Its worst case space complexity is very high however. It has O(n*k) space complexity where n is the amount of internal nodes (one for each unique character inserted in one position) and k is the size of the alphabet.

Template Parameters
TThe char type to determine the kind of string to use, e.g. char for std::basic_string<char> (which is std::string).
